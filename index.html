    Link TPL

    A templating engine in PHP, using a XML syntax like, and a parser based on PCREs.

    Requires at least PHP 5.2.1, because of the utilisation of the Standard PHP Library and several object specifications available only since 5.2.1. Compatible PHP 5.3 and 5.4.

    Documentation

    A recent version of the documentation (in english) can be found in the docs folder, and an online version in available on Read the docs.org.

    For Link < 1.13 (aka Talus TPL), you may find an english documentation on Pingax's fork of Link TPL.

    Installation

    You have multiple ways to install Link. If you are unsure what to do, go with the tarball.

    From the tarball release

    1. Download the most recent tarball from the download page
    2. Unpack the tarball
    3. Move the files somewhere in your project

    Development version

    1. Install Git
    2. git clone git://github.com/Taluu/Link-TPL.git

    Via Composer

    1. Install composer in your project: curl -s http://getcomposer.org/installer | php

    2. Create a composer.json file in your project root:

        {
    "require": {
      "Taluu/Link-TPL": "1.13.*"
    }
  }

    3. Install via composer : php composer.phar install

    About this project...

    This project was initiated as a fork "from scratch" from the templating engine of the Fire Soft Board project. It also has some stuff inspired from the Twig's, PHPBB3's and Django's templating engines.
